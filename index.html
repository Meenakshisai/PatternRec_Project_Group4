🍇 CSE 455/555 Term Project Report

🧠 Visual Classification of Blackberry and Lime using Transfer Learning

Team Role: CV Engineer – Meenakshi Gundluri

📌 1. Introduction

This project focuses on visually identifying fruits—specifically blackberries and limes—and classifying them further by their presentation. Using deep learning and transfer learning, we build a smart classifier and deploy it for interactive use.

🗃️ 2. Dataset Preparation

📷 2.1 Image Collection

Images were captured manually to simulate real-world variations. They included cluttered backgrounds, natural light, and multiple contexts to ensure model robustness.

🌀 2.2 Data Variations

  • Halved
  • In Context
  • In a Container
  • Single Berry
  • Small Group
  • Whole

🗂️ 2.3 Folder Structure & Renaming

Images were renamed uniformly and sorted into train/val/test sets with an approximate 70:20:10 split. This was automated to ensure reproducibility.

🛠️ 3. Model Development

💻 3.1 Tools

  • Python, PyTorch, Torchvision
  • VS Code Jupyter Notebooks
  • Gradio & Hugging Face Spaces

⚙️ 3.2 Architecture

We fine-tuned a ResNet-18 pre-trained model. The final fully-connected layer was replaced to output either 2 or 6 classes based on the classification task.

🎯 3.3 Training Parameters

Loss: CrossEntropyLoss
Optimizer: Adam
Epochs: 5
